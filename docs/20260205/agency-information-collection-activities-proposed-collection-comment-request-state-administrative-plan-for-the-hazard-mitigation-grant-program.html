Agency Information Collection Activities: Proposed Collection; Comment Request; State Administrative Plan for the Hazard Mitigation Grant Program

Published: 2026-02-05 11:15:55
The Federal Emergency Management Agency, as part of its continuing effort to reduce paperwork and respondent burden, invites the general public to take this opportunity to comment on an extension, without change, of a currently approved information collection. In accordance with the Paperwork Reduction Act of 1995, this notice seeks comments concerning the State Administrative Plan for the procedural guide that details how the State will administer the Hazard Mitigation Grant Program (HMGP).
